# EasyAPI
ตัวอย่าง API ง่ายๆ
## วิธีการติดตั้ง
Requirement :  `Node.js` [**Download**](https://nodejs.org/en)

```bat
  npm install
```
---
HTTP กำหนดรหัสสถานะมาตรฐานเหล่านี้ซึ่งสามารถใช้เพื่อถ่ายทอดผลลัพธ์ของคำขอของลูกค้า รหัสสถานะแบ่งออกเป็นห้าประเภท
